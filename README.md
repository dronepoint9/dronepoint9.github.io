# Hub

Personal app launcher for Tulane Construction Group / DronePoint tools.

Live at: `https://yourusername.github.io`

## Adding a new app

1. Build the app in its own repo (e.g. `github.com/yourusername/new-app`)
2. Enable GitHub Pages on that repo
3. Add a card to `index.html` in this hub repo pointing to its URL
4. Remove the `coming-soon` class when it's ready

## Repo setup

This repo must be named **`yourusername.github.io`** exactly — that's what makes it serve at the root URL with no subfolder.
